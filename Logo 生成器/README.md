# Logo 生成器

**简介**：用户输入品牌需求，多个大模型各自根据需求绘制 SVG 格式的 Logo 设计，然后让一个大模型做评分员从多个方面对备选作品进行评分，最终选出优胜作品。

**备注**：Dify 的 SVG 渲染效果不佳，如遇渲染出错，可以用 [SVGviewer](https://www.svgviewer.dev/) 预览
